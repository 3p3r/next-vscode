# next-vscode
 
VSCode-web embedded in a Next.js app.

Use this as a framework to build web applications that require code editing, debugging, and other VSCode features alongside custom web functionalities.

## running

```sh
npm install
npm run build
npm run dev
```

Then open http://localhost:3000 in your browser.

## deploying

You can use <https://opennext.js.org> in combination with <https://www.npmjs.com/package/cdk-nextjs-standalone> to deploy this app to AWS Lambda serverless.
